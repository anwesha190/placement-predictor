# PlacementIQ

PlacementIQ is an AI-powered student placement prediction and career guidance platform built using Python, Streamlit, Scikit-learn, Pandas, NumPy, and Plotly.

The application analyzes key academic and professional indicators such as CGPA, project experience, internships, technical skills, and communication abilities to estimate placement probability and provide personalized career recommendations.

## Features

* Placement probability prediction using Machine Learning
* Career path recommendations
* Interactive and user-friendly dashboard
* Data visualization using Plotly
* Personalized improvement suggestions
* Real-time prediction results

## Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Plotly

## Use Cases

* Students preparing for campus placements
* Career planning and skill assessment
* Academic and placement analytics
* Data science and machine learning portfolio demonstration

## Getting Started

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
